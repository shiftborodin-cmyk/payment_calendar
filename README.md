# Платёжный календарь

Мобильное приложение для планирования личных доходов, расходов и будущего остатка денег. Всё необходимое — в календаре, списке платежей и понятном прогнозе по дням.

> Проект находится в активной разработке. Данные хранятся локально на устройстве, поэтому приложение можно использовать без обязательной регистрации и облачной синхронизации.

## Возможности

- календарь в режимах «Месяц» и «Неделя»;
- платежи и доходы с категориями, статусами и повторением;
- оплата отдельного вхождения повторяющегося платежа;
- быстрый переход к ближайшему платежу и его оплата с главной страницы;
- прогноз остатка денег по дням месяца;
- интерактивный график с датами и курсором;
- мягкая подсветка дней с предполагаемой нехваткой денег;
- тёмная и светлая темы;
- русский и английский языки;
- настройка имени пользователя и цветового акцента;
- Android APK для установки на телефон.

## Данные и приватность

Платежи и категории хранятся локально через AsyncStorage и не отправляются на сервер. Supabase используется только для авторизации пользователя. Синхронизация между устройствами пока не включена.

## Попробовать приложение

Для установки готового APK откройте [страницу релизов](https://github.com/shiftborodin-cmyk/payment_calendar/releases) или [текущую сборку в Expo](https://expo.dev/accounts/jugeniy/projects/payment-calendar/builds/55f5025e-cbc7-448c-ae3a-876ba43dd409).

Для запуска в режиме разработки понадобятся Node.js LTS, npm и Git:

```bash
npm install
npx expo start -c
```

После запуска проект можно открыть через Expo Go. Для обычного использования приложение собирается в APK через EAS Build.

## Технологии

- React Native + Expo;
- TypeScript;
- Expo Router;
- AsyncStorage для локальных платежей и категорий;
- Supabase Auth;
- EAS Build для Android APK.
