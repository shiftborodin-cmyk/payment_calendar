# Платёжный календарь

Стабильный MVP мобильного приложения для планирования личных расходов и доходов.

Текущая стабильная версия: **0.1.1**.

## Что умеет приложение

- календарь в режимах «Месяц» и «Неделя»;
- платежи и доходы с повторением, статусами и категориями;
- локальное хранение платежей и категорий на устройстве;
- прогноз остатка денег по дням месяца;
- мягкая подсветка дней с предполагаемой нехваткой денег;
- тёмная и светлая темы;
- русский и английский языки;
- настройка имени пользователя и цветового акцента;
- сборка Android-приложения в формате APK.

## Важное о данных

Платежи и категории сейчас хранятся локально через AsyncStorage и не отправляются на сервер.
Supabase используется только для авторизации пользователя. Синхронизация между устройствами пока не включена.

## Как запустить проект разработчику

Понадобятся:

- Node.js LTS и npm;
- Cursor или Visual Studio Code;
- Git;
- файл `.env` с переменными из `.env.example` для авторизации Supabase.

Установка и запуск:

```bash
npm install
npx expo start -c
```

Для быстрой проверки во время разработки можно использовать Expo Go. Для обычного использования приложение собирается в APK через EAS Build.

## Технологии

- React Native + Expo;
- TypeScript;
- Expo Router;
- AsyncStorage для локальных платежей и категорий;
- Supabase Auth;
- EAS Build для Android APK.
