# Платёжный календарь

MVP мобильного приложения для личных платежей, задач и финансовых обязательств.

## Что понадобится для запуска

- Node.js LTS: программа, которая позволяет запускать проект и устанавливать зависимости.
- npm: устанавливается вместе с Node.js.
- Expo Go на телефоне: приложение для быстрого просмотра мобильного проекта без публикации в стор.
- Аккаунт Supabase: нужен для регистрации, входа и хранения данных.
- Редактор кода: подойдёт Visual Studio Code или Cursor.
- Git: полезен для сохранения истории изменений.

## Текущий стек

- React Native + Expo
- TypeScript
- Expo Router
- Supabase Auth + PostgreSQL
- expo-notifications
